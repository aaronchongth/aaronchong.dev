VEGAN - Visual Enhancement GAN

Visual Enhancement GAN is a generative adversarial network trained to deblur images to be used as data for high speed applications, specifically in the self driving domain. We built on the work of Kupyn et. al to improve multiscale performance, introduced training with Scaled Perceptual Loss, Smoothness Loss, as well as using a Random Scaled Crop augmentation technique. Additionally we also introduced a Fourier Domain Error, as an way to better evaluate the performance of an image deblurring system.
